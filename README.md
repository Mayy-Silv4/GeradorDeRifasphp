# 🎟️ Gerador de Rifas em PHP

Projeto simples desenvolvido em PHP para gerar números de rifas de forma automática, permitindo criar listas organizadas para sorteios.

---

## 🚀 Funcionalidades

* Gerar números de rifa automaticamente
* Definir quantidade de números desejados
* Exibir lista organizada de rifas
* Interface simples e intuitiva
* Possibilidade de reutilizar para sorteios

---

## 🛠️ Tecnologias utilizadas

* PHP
* HTML
* CSS

---

## ⚙️ Como funciona

O usuário informa a quantidade de números que deseja gerar.
O sistema cria automaticamente uma sequência numérica, por exemplo:

* De 1 até 50
* De 1 até 100
* Ou qualquer quantidade definida

Esses números são exibidos na tela de forma organizada.

---

## 📁 Estrutura do projeto

```plaintext
/gerador-rifas
│── index.php
│── style.css
```

---

## ▶️ Como executar

1. Coloque a pasta dentro do `htdocs` (XAMPP ou similar)
2. Inicie o Apache
3. Acesse no navegador:

```plaintext
http://localhost/gerador-rifas
```
